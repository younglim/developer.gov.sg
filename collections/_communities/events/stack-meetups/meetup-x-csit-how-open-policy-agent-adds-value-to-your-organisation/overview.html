Overview

Who should attend: If you have an interest in API Security and microservices, this Meetup is for you!

Want to learn how Open Policy Agent (OPA) benefits an organisation? Join us for this STACK-X Meetup, co-organised with the Centre for Strategic Infocomm Technologies (CSIT), to discover more.

Speakers from CSIT and GovTech will share on how OPA is being implemented, how Envoy and OPA proved invaluable in CSIT's journey in breaking up monoliths into microservices, and how the Whole-of-Government Platform-as-a-Service (PaaS) makes use of OPA (Gatekeeper) to enforce a variety of requirements.
